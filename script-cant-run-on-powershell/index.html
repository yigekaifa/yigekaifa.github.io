PowerShell禁止执行脚本的解决方法

如果出现类似下面的提示：

解决方法：

运行Get-ExecutionPolicy。如果返回Restricted，然后运行set-ExecutionPolicy RemoteSigned

然后就可以运行命令了。

另外如果还不能执行，例如你npm安装的一些命令不能运行，那么请你把C:\Users\用户名\AppData\Roaming\npm添加到系统变量Path里面。
